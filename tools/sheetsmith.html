    Format, fitting, and safe style guide

    Top-level fields: version: 1, layout (playing-9 or business-10), mode (collection or repeat), theme (classic, modern, or minimal), optional accent, and fit.

    Blocks are title/header, subheader, paragraph, footer, image, divider, and spacer. Image fit is only contain or cover. Themes, accents, spacers, and fits are controlled tokens—HTML and CSS are never interpreted.

    fit: auto reduces body text from 12px to a documented minimum of 8px and headings from 22px to 12px. Any remaining overflow is outlined and warned, never silently truncated. fit: strict never resizes and remains available.

    Browser-local image helper

    Choose a PNG, JPEG, WebP, GIF, or SVG up to 2 MiB. The resulting data URL can be copied into an image block’s src. Nothing is uploaded.

    Private by design: processing is fully browser-local. No account, server, upload, storage, cookies, or network request is required. HTTPS images load directly from their host only when you explicitly include one.
